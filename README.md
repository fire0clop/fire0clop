### About

I design and build **complete mobile products** — from the FastAPI backend and its
database, auth, realtime layer and infrastructure, all the way to a polished native
iOS/macOS client. Comfortable owning the whole stack: WebSockets, WebRTC calling,
push notifications, geospatial search, packet-tunnel VPNs and server-authoritative
game engines.

- 🧩 **Backend:** FastAPI · async SQLAlchemy 2.0 · PostgreSQL · Redis · Alembic · Celery/ARQ · WebSockets · JWT
- 📱 **Apple:** Swift · SwiftUI · Combine · NetworkExtension · WebRTC · CallKit · APNs
- 🛠 **Infra:** Docker · GitHub Actions · S3/MinIO · Nginx/Gunicorn
